# Project LoNA | Liquid optimised Neural Architecture

## Overview

Project LoNA: High-intelligence liquid neural networks for tiny models. Optimized for small-scale compute without compromising on architectural depth.

## Building

Requires CMake.

```bash
cd build && cmake .. && make
```

## Status

**Pre-alpha** — Project scaffolding. Core architecture not yet implemented.

Planned areas:
- Architecture (core model structure, neural design)
- Training (pipeline, optimisation, loss functions)
- Benchmarks (performance testing, evaluation metrics)
